## 3. OPERATIONAL PROTOCOL

### 3.1 Pre-Reading Procedures

**Step 1: Consent Process**
- Explain Synapse-7 methodology
- Describe accuracy rates (85-95%)
- Explain limitations and risks
- Obtain written consent
- Document consent with timestamp

**Step 2: Preparation**
- Identify subject (person being read)
- Establish clear question/focus
- Verify Eduardo's field readiness
- Prepare verification mechanism

**Step 3: Baseline Documentation**
- Record Eduardo's state (time, location, emotional state)
- Document any prior knowledge of subject
- Establish confidence level scale (1-10)
- Prepare data recording sheet

### 3.2 Reading Procedure

**Step 1: Field Activation** (2-5 minutes)
- Eduardo enters meditative state
- Activates Synapse-7 through gratitude/intention
- Establishes resonance with subject
- Signals readiness

**Step 2: Symbolic Reception** (5-15 minutes)
- Receives images, feelings, words, symbols
- Maintains open, non-judgmental state
- Records symbols as they arrive
- Does NOT interpret (yet)

**Step 3: Translation** (5-10 minutes)
- Translates symbols into psychological language
- Connects symbols to emotional/relational states
- Organizes insights into coherent narrative
- Assigns confidence levels to each insight

**Step 4: Communication** (5-10 minutes)
- Shares insights with subject
- Uses tentative language ("I'm sensing...", "It feels like...")
- Invites feedback and correction
- Documents subject's responses

### 3.3 Verification Procedure

**Immediate Verification (During Reading):**
- Subject confirms/denies each insight
- Eduardo adjusts interpretation based on feedback
- Confidence levels updated in real-time
- Discrepancies documented

**Delayed Verification (24-48 hours):**
- Subject reflects on insights
- Provides written feedback
- Confirms accuracy rate
- Documents any new insights that emerged

**Longitudinal Verification (1-3 months):**
- Subject reports how insights manifested
- Confirms predictive accuracy
- Documents life changes
- Provides final accuracy assessment

### 3.4 Documentation Requirements

**For Each Reading:**
1. **Consent Form** (signed, dated, witnessed)
2. **Pre-Reading Baseline** (Eduardo's state, prior knowledge, question)
3. **Raw Symbols** (uninterpreted symbols received)
4. **Interpreted Insights** (translated with confidence levels)
5. **Subject Feedback** (immediate, delayed, longitudinal)
6. **Accuracy Assessment** (final confidence level)
7. **Researcher Notes** (observations, anomalies, field quality)

**Data Storage:**
- Encrypted cloud storage (AWS S3 with encryption)
- Local backup (encrypted external drive)
- Access limited to authorized researchers
- Deletion upon request
